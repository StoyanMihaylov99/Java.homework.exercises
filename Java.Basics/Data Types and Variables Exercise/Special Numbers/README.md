Special Number:

A number is special when its sum of digits is 5, 7, or 11.
Write a program to read an integer n and for all numbers in the range 1…n to print the number and if it is special or not (True / False).



Example:

Input:                  Output:

15                     1 -> False
                       2 -> False
                       3 -> False
                       4 -> False
                       5 -> True
                       6 -> False
                       7 -> True
                       8 -> False
                       9 -> False
